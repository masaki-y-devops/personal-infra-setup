## 使用例

~~~
cd c:\Users\%username%\Desktop
sudo .\AddTasks-SetAdguardDoH.bat
~~~

## 作成の経緯

近年、悪質なフィッシング広告やマルウェアを含んだサイトに誘導する広告が増えているため、エンドユーザーとしては、そのような広告をブロックすることで、自分のパソコンのセキュリティを維持する必要も生じていると思う。

設定を何度も行うのが大変なため、Windows上のDNS over HTTPS設定をワンストップで作成するスクリプトにした。

## 選定理由

フォールバック先としてControl-D社のDNSサービスも検討したが、しばらく使ってみてAdGuard社のサービスが安定していたので、同社のアドレスを指定した。

## 課題点

システム上のすべて（Tailscale以外）のDNS名前解決に対し、AdguardパブリックDNSを使用する設定となっている。

そのため、プロバイダ提供のDNSサーバーではない関係上、アクセスできないサイトが生じたり、サイトレイアウトが崩れる可能性があると思われる。
