## 使用例

管理者権限が必要です。

sudo（設定 -> 詳細設定 -> sudoの有効化)を使用している場合、スクリプトを配置したディレクトリに移動して、実行します。

デスクトップ直下にスクリプトがあり、コマンドプロンプトの場合:

~~~
cd C:\Users\%username%\Desktop
sudo .\InitWinSetup.bat
~~~

## 作成経緯

Linuxだけでなく、ゲームをするためにWindowsを使用することも多いので、再セットアップをすぐ行うために、自分用にWindowsの内蔵アプリ、サービス、電源管理、新規タスクの追加を実行しています。

## 選定理由

- Windowsクリーンインストール直後のまっさらの状態からでも実行可能で、実行環境の事前セットアップが不要であるため。

- Windows固有の設定（UWPアプリの削除やサービスの停止）がほとんどなので、互換性の低さはある程度許容できると考えたため。
