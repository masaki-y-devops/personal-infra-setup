## 使用例

本スクリプトをタスクスケジューラに登録してログイン時に実行。

クラウドストレージ上にスクリプトを置き、同期クライアントの起動を待って（3分と想定）実行する場合:

~~~
schtasks /create /tn UpdateUserApps /tr "K:\Scripts\Windows\Tasks\UpdateUserApps.bat" /sc onlogon /delay 0003:00 /f
~~~

## 作成経緯

Windows環境において、付属の自動更新機能がない、あるいは独自にバイナリを管理したいツール（rclone, 7-zip等）がある。

その更新作業は手動に頼らざるを得ず、「更新漏れによるセキュリティリスク」と「毎回繰り返しのチェック作業の煩雑さ」が課題。

本スクリプトは、これらのツールを一括で最新の状態に保つために作成。

## 選定理由

確かに7-zipやrcloneはchocolateyパッケージがある。

しかし、パッケージング作業という中間工程がある関係上、公式サイトやGitHubから直接実行ファイルをダウンロードするよりも不確実性リスクが大きいと判断。

また、chocolateyやwingetにはサービス終了のリスクもゼロではないと考え、長期運用には不安が残る可能性があるため、上流から直接引っ張ってくる選択肢を選んだ。
