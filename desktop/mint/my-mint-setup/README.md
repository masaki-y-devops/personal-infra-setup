概要
===

Linux Mintクリーンインストール後に環境を自分用に調整するために書きました。

使用例
=====

スクリプトをカレントディレクトリにコピーし、

~~~bash
sudo ./MyMintSetup.sh
~~~

で実行します。

主なトラブルシューティング・課題
=============================

## GPDデバイス判定の限界

LinuxMint自体は手動でUSBブートしてインストールしているため、その過程で```/etc/hostname```を設定します。

スクリプト内では```/etc/hostname```の内容で動作を変えているため、自分のセットアップ時のデバイス名命名ミスがあると処理が破綻してしまいかねないのが問題点です。

## if構文か、case構文か？

処理の条件分岐にとりあえず「上から下にチェックする」というわかりやすさ、書きやすさを優先して、if構文で構成してみましたが、case構文のほうが評価・チェック頻度の関係で動作が速くなるかもしれません。
